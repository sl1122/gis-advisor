# 缓冲区分析 - ArcGIS Pro

## 场景

求道路、河流、设施点一定距离范围内的对象，或生成约束区、影响区。

## 前置检查

- 输入图层必须使用投影坐标系。
- 明确缓冲距离和单位。
- 明确是否需要融合缓冲区。

## 操作步骤

1. 打开 `分析 > 工具`。
2. 搜索 `缓冲区 Buffer`。
3. 输入要素选择目标图层。
4. 设置距离，例如 `500 Meters`。
5. 设置融合类型：
   - 需要整体影响区：选择融合。
   - 需要保留每个对象影响区：不融合。
6. 运行后打开属性表和地图检查结果。

## 参数表

| 参数 | 建议 | 说明 |
| --- | --- | --- |
| 输入要素 | 点/线/面图层 | 如道路、学校、医院 |
| 距离 | 题目给定值 | 必须确认单位 |
| 融合 | 按题意选择 | 影响区通常融合，逐对象分析通常不融合 |
| 输出 | `buffer_xxx.gpkg/shp` | 文件名应表达距离和对象 |

## 常见错误

- 在经纬度坐标系下直接缓冲。
- 忘记融合导致面积重复统计。
- 缓冲对象选错，例如把建筑缓冲而不是道路缓冲。

## 检查清单

- 距离单位正确。
- 缓冲范围与原图层位置一致。
- 后续叠加或筛选前确认输出字段和几何有效。
