# 缓冲区分析 - QGIS

## 场景

使用免费 QGIS 复现 ArcGIS Pro 的缓冲区分析。

## 前置检查

- 图层坐标系应为米制投影坐标系。
- 如果是经纬度坐标系，先使用 `重新投影图层`。

## 操作步骤

1. 打开 `处理工具箱`。
2. 搜索 `缓冲区 Buffer`。
3. 选择输入图层。
4. 设置距离。
5. 设置线段数，常用 `5` 到 `8`。
6. 按题目决定是否勾选融合。
7. 运行后检查输出图层。

## 参数表

| 参数 | 建议 | 说明 |
| --- | --- | --- |
| 输入图层 | 目标要素 | 点、线、面均可 |
| 距离 | 题目给定值 | 单位来自图层 CRS |
| 线段数 | 5 或 8 | 越大越圆滑，文件越大 |
| 融合 | 按题意 | 避免重复面积时常需要融合 |

## 常见错误

- 地理坐标系下距离单位变成度。
- 忽略无效几何。
- 多图层叠加前没有统一 CRS。

## 检查清单

- 输出位置正确。
- 面积/长度字段在缓冲后重新计算。
- 后续按位置选择时确认谓词是相交、包含还是范围内。
