# GIS 题型分类

## 1. 缓冲区与邻近分析

常见关键词：缓冲、距离、范围内、道路影响、噪声影响、500m、1km。

典型任务：

- 求某类要素一定距离范围内的对象。
- 根据距离筛选建筑、学校、医院、道路影响区。
- 生成约束区或排除区。

风险：

- 未使用投影坐标系导致距离错误。
- 缓冲距离单位不明确。
- 缓冲后未融合或错误融合。

## 2. 叠加分析与面积统计

常见关键词：相交、擦除、裁剪、融合、面积、空地率、覆盖率。

典型任务：

- 片区面积、建筑覆盖面积、空地率。
- 约束区排除和候选区提取。
- 多图层叠加后按区域汇总。

风险：

- 擦除顺序反了。
- 叠加后未重新计算面积。
- 多部件要素未拆分导致统计错误。

## 3. 栅格计算与重分类

常见关键词：重分类、栅格计算、NoData、指数、阈值、变化检测。

典型任务：

- 土地利用类别提取。
- TCI、NDVI、NDBI、TWI 等指数计算。
- 变化编码和转移矩阵。

风险：

- NoData 传播未检查。
- 离散栅格重采样方式错误。
- 公式里的单位和对数类型未确认。

## 4. 水文分析

常见关键词：填洼、流向、汇流累积、河网、流域、集水面积。

典型任务：

- DEM 填洼。
- 流向和流量累积。
- 阈值提取河网。
- 流域或出水口分析。

风险：

- 未填洼直接做流向。
- 汇流阈值没有按像元面积换算。
- DEM 垂直单位与水平单位不一致。

## 5. 分区统计与指标回连

常见关键词：分区统计、乡镇、行政区、均值、总和、GDP、夜间灯光。

典型任务：

- 按乡镇统计栅格均值或总和。
- 将统计表回连到行政区。
- 用夜间灯光权重分配 GDP。

风险：

- 分区字段不唯一。
- 回连字段类型不一致。
- Null 值未检查。

## 6. 网络分析

常见关键词：网络分析、最短路径、最近设施、服务区、OD 矩阵、道路网络、通行成本。

典型任务：

- 最近设施点。
- 服务区分析。
- OD 成本矩阵。
- 可达性分析。

风险：

- 不能因为出现“道路”就判断为网络分析。
- 道路网络必须连通。
- 成本字段、方向、限制条件必须明确。

## 7. 制图输出

常见关键词：专题图、图例、比例尺、指北针、布局、颜色、标注。

典型任务：

- 分类图、连续栅格图、统计专题图。
- 导出 PDF/PNG。
- 检查图例和单位。

风险：

- 连续数据用了分类色带，离散数据用了拉伸色带。
- 图例没有解释编码。
- 缺少比例尺、单位、指北针或图名。
