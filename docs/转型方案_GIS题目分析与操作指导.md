# GIS 题目分析与操作指导系统转型方案

## 新定位

本项目不再定位为“一键自动完成 GIS 数据处理”的软件，而是定位为：

> GIS 题目分析与操作指导系统。

核心目标是帮助用户理解模糊竞赛题、作业题和科研项目需求，拆解任务、识别数据、推荐 ArcGIS Pro 优先和 QGIS 免费替代的操作路线，并给出风险、确认点和结果检查清单。

## 功能优先级

1. 题目拆解与项目需求分析
2. 训练反思和相似案例匹配
3. ArcGIS Pro / QGIS 双路线中文操作指导
4. 缺失条件、风险和人工确认点提示
5. 结果检查清单
6. 流程树和操作卡片展示
7. 可选自动执行确定性步骤

## 不再承诺

- 不承诺输入文件夹后一键得到正确答案。
- 不承诺自动完成补绘、字段含义判断、制图表达和人工判读。
- 不承诺 QGIS 自动执行结果等价于 ArcGIS Pro 竞赛标准结果。

## 自动执行边界

可以尝试自动执行：

- 坡度、坡向、山体阴影
- 填洼、汇流累积、阈值提取河网
- 缓冲、裁剪、差集
- 明确公式的栅格计算
- 字段和参数明确的分区统计

只提供指导或半自动：

- 补绘建筑或其他要素
- 字段含义判断
- 图例和符号系统最终定稿
- 多候选点批量可视域
- 建筑日照和 3D 太阳阴影判读
- 权重不明确的多因子适宜性评价

## 默认软件路线

默认路线：

- ArcGIS Pro 优先
- QGIS 免费替代
- QGIS / GDAL / Whitebox 仅作为可复现和可自动执行后端

操作路径显示格式：

```text
ArcGIS Pro：分析 > 工具 > 邻近分析 > 缓冲区（Buffer）
QGIS 替代：处理工具箱 > 矢量几何 > 缓冲区（Buffer）
```

## 训练反思使用方式

训练反思应作为动态知识库，而不是固定死规则。

每条知识应尽量结构化为：

- 原文
- 题型
- 数据类型
- 操作链
- 参数经验
- 常见错误
- 结果检查
- 相似案例
- 差异点

重复知识需要合并；相似但不同的知识需要标注差异。
