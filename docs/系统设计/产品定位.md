# 产品定位

GIS Advisor 是一个 GIS 题目分析与操作指导系统。

## 不做什么

- 不承诺一键自动得出竞赛答案。
- 不替代 ArcGIS Pro、QGIS、ENVI 等专业软件。
- 不在参数不明确时伪造阈值、字段或权重。

## 做什么

- 读取题目和项目文件夹。
- 区分题目、数据、表格、说明文档。
- 拆解题型、识别公式和数据需求。
- 推荐 ArcGIS Pro / QGIS 双路线。
- 标出风险、缺失项和人工确认点。
- 对表格和部分确定性步骤提供核查或自动处理。

## 模块

- 文件夹扫描与数据识别。
- PDF/文档题面提取。
- AI/规则题目分析。
- 训练反思索引匹配。
- 操作模块库。
- 辅助流程生成。
- 表格统计核查。
- 可选 GIS 后端执行。

## 成功标准

- 用户能更快看懂题目。
- 用户知道应该用哪些工具、哪些参数要确认。
- 系统不把模糊步骤误执行成错误结果。
- 结果检查更清楚，减少竞赛和作业中的低级错误。
